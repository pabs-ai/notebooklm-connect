# Changelog

All notable changes to NotebookLM Connect will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-26

### Added
- Initial release
- One-click content extraction from web pages
- Right-click context menu integration
- Smart content cleaning (removes ads, navigation, footer)
- Source attribution (URL, title, author, timestamp)
- Text selection support (send specific paragraphs)
- Usage statistics tracking
- Extension popup with stats display
- NotebookLM helper popup with copy button
- Chrome badge notifications (success/error)

### Features
- Manifest V3 compliance
- Zero external dependencies
- Local-only data storage
- Privacy-focused design

## [Unreleased]
